Guia do Pixel Tracker JS
1. Onde instalar o snippet?
Cole o script no <head> de todas as landing pages que recebem tráfego dos seus links do AppLink. Veja qual seu caso:
📄 HTML puro / Webflow / Framer
Cola direto no <head> do template
🛒 Shopify
Theme → Edit code → theme.liquid → <head>
📝 WordPress
Plugin "Insert Headers and Footers" → Scripts in Header
📊 Google Tag Manager
Tag custom HTML → Trigger All Pages
🌐 Wix
Settings → Tracking & Analytics → Custom
🎯 Outras (Squarespace, Hubspot…)
Procure por "Custom HTML in <head>" na config
2. Copie o snippet abaixo
Já está com o seu workspace ID WK_a8f2c3 preenchido. Cole no <head> antes de qualquer outro pixel.
<!-- AppLink Pixel Tracker -->
<script
  src="https://app.applink.io/pixel/WK_a8f2c3.js"
  data-workspace="WK_a8f2c3"
  async></script>
  • O script é ~3KB gzipped e async — não bloqueia o carregamento
  • Auto-detect de click_id da URL ao receber tráfego do AppLink
  • 1st party cookie → bypassa iOS ATT e ad blockers
3. Verificar se está rodando
Após colar o snippet e publicar a LP, confirme com 1 click:
Auto-check (recomendado)
Volte ao AppLink → tela Pixels → clique no botão "Testar instalação agora" dentro do card "Pixel Tracker JS".
O sistema visita seu domínio, dispara um evento de teste e confirma em ~5 segundos. Status do domínio atualiza pra ● Instalado.
⚠ Domínio precisa estar autorizado. Adicione o domínio da LP em "Domínios autorizados deste workspace" antes — caso contrário o snippet é bloqueado por CORS.
4. Smart Tracking — eventos automáticos
O snippet já detecta automaticamente. Não precisa código extra se sua LP tem <form> padrão.
  • Form fill: escuta submit em qualquer <form>. Hash automático de email/phone pra Match Quality 9+/10 no Meta.
  • Tempo de página: dispara WARM quando o usuário passa 30s na LP (configurável no link).
  • Scroll depth: dispara WARM quando passa de 80% (configurável).
  • Custom events (avançado): chame applink.qualify() ou applink.pure() em momentos específicos do funil (clicou em CTA, agendou call, etc.)
// Exemplo: marcar lead como PURE quando clica em "Agendar"
document.querySelector('.cta-agendar').addEventListener('click', () => {
  applink.pure({ value: 500, currency: 'BRL' });
});
Pronto pra rodar! 🚀
Depois de instalar, os primeiros eventos chegam em ~2-5 min. O Match Quality leva 24h pra refletir no Meta Ads Manager.
